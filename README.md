# CivicVoice

![Status](https://img.shields.io/badge/Status-In%20Development-2563EB)

A Flutter-based civic issue reporting platform designed to help citizens report community issues while enabling municipal authorities to efficiently manage, assign, and resolve them.

## Project Overview

CivicVoice is a semester project for Mobile Computing & Mobile Application Development.

The platform allows citizens to report civic issues such as:

- Potholes
- Flooding
- Blocked drains
- Broken streetlights
- Waste disposal issues
- Water leaks
- Damaged public infrastructure

Reports include location, photos, descriptions, and status tracking throughout their lifecycle.
